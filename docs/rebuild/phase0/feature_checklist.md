# Feature Checklist (Parity Baseline)

## Functional Areas
- Identity and access bootstrap
- RBAC and scope enforcement
- Vendor and offering lifecycle workflows
- Projects and linked entities
- Imports (single and bundle workflows)
- Pending approvals workflow
- Reports run/download/email request
- Help center and feedback
- Health and diagnostics endpoints

## Rebuild Rule
Each area above must have parity acceptance tests in the new stack before cutover.

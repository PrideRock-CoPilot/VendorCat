# Phase 5 Feature Checklist

## Import Job Parity
- Import job create/list/get/update endpoints
- Import job status tracking (submitted, processing, completed, failed)
- File format detection (CSV, JSON, XML, delimited)
- Mapping profile management (create, list, apply)
- Import preview/validation step
- Source-to-target field mapping
- Import job UI list/detail views

## Workflow Decision Parity  
- Workflow decision create/list/get endpoints
- Workflow decision status tracking (pending, approved, rejected)
- Decision audit trail
- Workflow UI list/detail views

## Tests
- Rebuild tests for import job API flows and UI pages
- Rebuild tests for workflow decision API flows
- Permission enforcement tests for import/workflow mutation routes
